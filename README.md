eventsourcing-with-kevin
========================

Play with eventsourcing.  For more info, you can see the appendix at the bottom.

### Getting started
    gem install bundler
    bundle

### Checking out an example
    ruby examples/echo_events.rb

### Running the tests
    rake test

### Appendix
